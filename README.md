# Spellwriters

This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 17.0.10.

## Getting Started

### Prerequisites

- Node 18.15+

### Installation

1. Clone the repo
1. `npm i`
1. `npm run gamedata`

## Development

1. `npm start`

## Building

1. `npm run build:all`
